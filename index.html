EXPLICAÇÃO E UTILIZAÇÃO DE TAGS

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas na internet. Ela é a base da estrutura de um site, definindo o conteúdo e a organização dos elementos.

Para que usamos HTML?

  • Estruturação de conteúdo (títulos, parágrafos, imagens, links).
  • Criação de formulários.
  • Definição de tabelas.
  • Inclusão de mídia (vídeos, áudios).
  • Criação de links entre páginas.

Principais tags HTML

  1. <h1> <h2> <h3> <h4> <h5> <h6> <h6>

    2. São tags de títulos, cada uma definindo o tamanho.
    Começando do maior, que é o h1 para o menor, que é o h6

    Exemplos

    <h1> <h1/>

    <h2> <h2/>

    <h3> <h3/>

    <h4> <h4/>

    <h5> <h5/>
    <h6> <h6/>

  2. <p>

    3. representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário

    Exemplo

    <p> </p>


  3. <mark>

    4. representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

    Exemplo

    <mark> </mark>

  4. <small>

    5. Coloca o texto em uma letra de menor tamanho.Também pode ser usado para marcar avisos obrigatórios, restrições legais ou de uso de conteúdo, como copyrights.

    Exemplo

    <small> </small>

  5. <i>

    6. representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico".

    Exemplo

    <i> </i>

  6. <u>

    7. tag representa algum texto que não é articulado e estilizado de forma diferente do texto normal, como palavras com erros de ortografia ou nomes próprios em texto chinês. O conteúdo dentro é normalmente exibido com um sublinhado.

    Exemplo

    <u> </u>

  7. <strong>

    8. é utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas). Podem ser sentenças que são de grande importância para toda a página, ou, você pode meramente tentar pontuar que algumas palavras são de grande importância, comparado ao conteúdo próximo.

    Exemplo

    <strong> </strong>

  8. <ol>

    9. representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

    Exemplo

    1. <li> </li>
    2. <li> </li>
    3. <li> </li>

  9. <ul>

    10. Representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes. Caracteristicamente, os itens em uma lista desordenada são exibidos com um marcador que pode ter várias formas, como um ponto, um círculo, ou um quadrado.

    Exemplo

    • <li> </li>
    • <li> </li>
    • <li> </li>


  10. <a>

    11. A tag que define um hiperlink, que é usado para vincular de uma página a outra.

    Exemplo

    <a href="https://www.w3schools.com">Visit W3Schools.com!</a>

  11. <hr>

    12. Representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML, representava uma linha horizontal. Pode continuar sendo exibida como uma linha horizontal nos navegadores, mas agora está definida em termos semânticos, em vez de termos de apresentação.

    Exemplo

    <hr1>

    <hr2>

    <hr3>


  12. <sub>

    13. é usado para identificar caracteres que devem ser renderizados em uma posição subscrita. O elemento deve ser usado para marcar texto de acordo com convenções tipográficas e não para propósitos estilísticos.

    Exemplo

    <sub> </sub>

  13. <sup>

    14. Especifica texto inline que deve ser exibido como sobrescrito por razões exclusivamente tipográficas. Sobrescritos são geralmente renderizados com uma linha de base elevada usando texto menor.

    Exemplo

    <sup> </sup>

  14. <blockquote>

    15. ou citação é utilizado para inserir um comentário de forma destacada com recuo de margem e espaçamento diante da linha anterior e diante da linha posterior.

    Exemplo

    <blockquote> </blockquote>

  15. <del>

    16. representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

    Exemplo

    <del>

  16. <abbr>

    17. representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela.

    Exemplo

    <abbr>
